# OBOMOCARE

A community-based organisation in Kisii and Nyamira counties, Kenya — delivering care and restoring dignity through food support, transport, personal care, companionship, and community action.

## Tech Stack

- React 18 + TypeScript
- Vite 5
- Tailwind CSS 4
- Firebase (auth, database, storage)
- Cloudinary (media)
- Framer Motion (animations)
- React Router v6
- Recharts (charts)
- Lucide React (icons)
